# Bem-vindo!

Esta é uma página de exemplo criada com o Sveltia CMS.

## Sobre este CMS

O Sveltia CMS é uma alternativa moderna e leve ao Netlify CMS, oferecendo:

- Interface intuitiva e moderna
- Suporte completo a Markdown
- Gerenciamento de mídia integrado
- Integração com GitHub
- Totalmente gratuito e open source

## Como usar

1. Acesse `/admin` no seu navegador
2. Faça login com sua conta do GitHub
3. Comece a criar e editar páginas!
